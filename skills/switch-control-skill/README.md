## Switch Control
Control a ble switch

## Description
Scan, discover, connect to a ble switch. control the switch and list out services that are broadcasting.

## Examples
 - "Turn on the switch"
 - "Turn off the switch"
 - "Power on"
 - "Power off"
 - "Switch on"
 - "Switch off"
 - "Turn on"
 - "Turn off"
